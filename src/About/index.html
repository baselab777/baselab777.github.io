연구실 소개

BASE Lab(Bidirectional AI-Software Engineering Lab)은 DGIST E3 전기전자컴퓨터공학(EECS) 소속 연구실로 김기섭 교수의 지도하에 인공지능(AI)과 소프트웨어 공학(SE)의 상호 발전에 대해 연구하고 있습니다. 우리의 연구는 인공지능으로 소프트웨어 공학의 문제를 해결하고, 소프트웨어 공학으로 더 나은 인공지능을 만드는 것에 기여합니다. 주요 연구 분야로는 AI4SE (AI for Software Engineering), SE4AI (Software Engineering for AI), Data4AI (Data for AI)가 있습니다. BASE Lab, affiliated with the Department of Electrical Engineering and Computer Science (EECS) at DGIST, conducts research on the mutual advancement of Artificial Intelligence (AI) and Software Engineering (SE) under the supervision of Prof. Kisub Kim. Our research contributes to solving problems in software engineering using AI, while also leveraging software engineering principles to develop better AI systems. Our main research areas include AI4SE (AI for Software Engineering), SE4AI (Software Engineering for AI), and Data4AI (Data for AI).

우리는 머신러닝, 딥러닝, 대규모 언어 모델(LLM)과 같은 인공지능 기술을 활용하여 코드 생성, 코드 인텔리전스, 코드 분석, 소스코드 분석, 프로그램 분석, 버그 탐지, 동적 분석, 소프트웨어 테스트, 자동 프로그램 수정(Automated Program Repair)과 같은 문제를 연구합니다. 또한 동시에 소프트웨어 공학적 방법론을 통해 더 신뢰할 수 있는 AI 시스템과 확장 가능한 AI 인프라를 구축하는 연구도 수행합니다. 이를 위해 소프트웨어 자동화, 소프트웨어 유지보수, 데이터셋 구축, AI 벤치마크, 데이터 품질, 머신러닝 시스템(ML Systems)과 같은 분야를 연구합니다. 개발자 생산성을 향상시키기 위한 AI 기반 프로그래밍 도구에도 관심을 가지고 있습니다. 여기에는 AI for Programming, 프로그래밍 언어 도구, 코드 분석 시스템, 소프트웨어 분석, 코드 인텔리전스 기술 등이 포함됩니다. BASE Lab은 이러한 연구를 통해 신뢰할 수 있는 AI 시스템과 지능형 소프트웨어 시스템을 구축하고 미래의 소프트웨어 개발 방식을 발전시키는 것을 목표로 합니다.SE4AI, Data4AI의 융합 연구를 통해 지능형 소프트웨어 시스템을 구축하고, 신뢰할 수 있는 인공지능을 구현하는 것을 목표로 합니다. We apply modern AI technologies such as machine learning, deep learning, and large language models (LLMs) to study software engineering problems including code generation, code intelligence, code analysis, source code analysis, program analysis, bug detection, dynamic analysis, software testing, and automated program repair. At the same time, we investigate how software engineering methodologies can be used to build reliable AI systems and scalable AI infrastructure. This includes research on software automation, software maintenance, dataset construction, AI benchmarks, data quality, and machine learning systems (ML systems). We are also interested in AI-powered programming tools that enhance developer productivity. This includes research on AI for programming, programming language tools, code analysis systems, software analytics, and code intelligence technologies. Through these efforts, BASE Lab aims to develop reliable AI systems and intelligent software systems that will advance the future of software development.
