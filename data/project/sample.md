# LLM 기반 자동화된 프로그램 수정

이 연구는 대규모 언어 모델(LLM)을 활용하여 소프트웨어의 버그를 자동으로 탐지하고 수정하는 것을 목표로 합니다.
기존의 프로그램 수정 기술(APR)은 정해진 패턴에 의존했지만, LLM은 코드의 문맥을 이해하여 보다 정교한 패치를 생성할 수 있습니다.

## 주요 연구 내용
- LLM의 파인튜닝 기법 연구
- 다중 언어 지원 패치 생성
- 수정된 코드의 신뢰성 검증

# LLM-based Automated Program Repair

This research aims to automatically detect and fix software bugs using Large Language Models (LLMs).
While traditional Automated Program Repair (APR) techniques relied on fixed patterns, LLMs can understand code context to generate more sophisticated patches.

## Key Research Areas
- LLM Fine-tuning techniques
- Multi-language patch generation
- Reliability verification of fixed code
